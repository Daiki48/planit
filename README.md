# Todo Management in CLI

# Feature

- ToDo Add
- ToDo Save
- ToDo Complete

# Usage

Create a `data` directory in advance.

```shell
$ cargo run -- add "Todo test1"
```

A `todo_list.json` is created in the `data` directory.
